Network Security Engineer
Bertanggung jawab untuk melindungi jaringan internal perusahaan atau kantor. Bertugas untuk mendeteksi, menyelidiki, dan mencegah gangguan yang mungkin dapat terjadi di sistem. Seorang network security engineer harus memastikan tidak adanya kebocoran data yang terjadi di jaringan.

Tugas-tugas lainnya antara lain:
1. Melakukan konfigurasi firewall di seluruh jaringan
2. Mendokumentasikan dan menentukan kebijakan keamanan data perusahaan
3. Merekomendasikan perubahan atau pembaruan dalam rangka perlindungan keamanan data perusahaan
Software Engineer
Bertanggung jawab untuk mengembangkan sistem, program, dan perangkat lunak dalam perusahaan. Software engineer dibutuhkan untuk menganalisis kebutuhan dan desain pengguna, konstruksi, serta uji aplikasi. Biasanya bekerja sama dengan software developer, programmer, dan spesialis quality control.

Tugas-tugas SE lainnya antara lain:
1. Merancang, membangun, dan mengetes sebuah sistem atau aplikasi
2. Mengelola aplikasi atau software secara berkala serta memperbaiki kesalahan yang terjadi
3. Merekomendasikan software update untuk aplikasi atau sistem yang sudah ada
UX Designer
Bertanggung jawab dalam menentukan apa pengalaman terbaik untuk pengguna dalam menggunakan suatu produk, misal website atau aplikasi. UX designer harus memastikan kegunaan, aksesibilitas, dan interaksi suatu aplikasi/situs menjadi user friendly.

Tugas-tugas UX Designer antara lain:
1. Melakukan riset pengguna dan target pasar
2. Membuat desain aplikasi sesuai hasil riset
3. Melakukan pengujian atau testing produk dari aplikasi yang sudah dibuat
Software Developer
Bertanggung jawab membangun serta menicptakan suatu produk. Sering bekerja sama dengan software engineer.

Tugas-tugas software developer antara lain:
1. mengembangkan dan mengarahkan pengujian sistem software dan prosedur validasi, pemrograman, dan dokumentasi
2. menganalisis kebutuhan pengguna dan kebutuhan software untuk menentukan kelayakan desain
3. bertemu klien dan project manager untuk merancang dan menciptakan sistem baru
Database Developer
Bertanggung jawab terhadap problem yang terdapat di dalam aplikasi yang berhubungan dengan database. Database developer berfokus untuk mengelola sistem database.

Tugas-tugas database developer antara lain:
1. Mengidentifikasi kebutuhan user dan mengelola database
2. Mencadangkan dan mengembalikan data untuk mencegah kehilangan data
3. Membuat dan melakukan tes terhadap modifikasi struktur database ketika diperlukan
Software Quality Assurance (QA) / Tester
Bertanggung jawab terhadap perencanaan jaminan kualitas aplikasi agar terhindar dari kegagalan pada saat dijalankan. Juga agar software tersebut dipastikan sudah berjalan dengan baik sebelum sampai ke tangan customer.

Tugas-tugas Software QA Tester antara lain:
1. Membuat program pengujian otomatis
2. Melakukan analisis dan membuat laporan pengujian
3. Memberikan feedback atas aplikasi yang diuji
Web Developer
Bertanggung jawab membuat dan mengembangkan situs. Selain itu web developer juga memastikan performa website optimal. Terdapat jenis-jenis web developer: back-end developer, front-end developer, dan full-stack developer.

Tugas-tugas web developer antara lain:
1. Merancang logika website dan membangun databasenya
2. Mendesain tampilan website
3. Mencari dan memperbaiki celah keamanan pada website
CRM Technical Developer
Bertanggung jawab terhadap pengembangan perangkat lunak yang mengimplementasikan perangkat tambahan untuk sistem CRM. Tidak jarang pula CRM Developer bertugas untuk merancang dan membuat modul tambahan, plugin khusus, atau rangkaian aplikasi perangkat lunak yang dapat diintegrasikan dengan sistem CRM.

Tugas-tugas CRM Developer antara lain:
1. Memperbaiki keluhan client terhadap sistem CRM yang dibuat
2. Mengumpulkan dan menginput data ke aplikasi seperti CMS (Content Management System), email responders, media sosial, google analytics, dll.
3. Mengidentifikasi dan menyelesaikan masalah yang ada pada sistem CRM
Technical Support
Bertanggung jawab untuk membantu menyelesaikan masalah yang dihadapi user. Masalah user bisa berupa keluhan maupun pertanyaan.

Tugas-tugas seorang technical support antara lain:
1. Berkomunikasi dengan user melalui layanan yang tersedia (hotline atau live chat)
2. Merekomendasikan solusi untuk memecahkan masalah
3. Membangun hubungan kerja dengan end-users
Systems Security Administrator
Bertanggung jawab terhadap keamanan jaringan komputer, termasuk keamanan dari sebuah administrasi sebuah perangkat seperti firewalls.

Tugas-tugas system security administrator antara lain:
1. Merancang dan mengidentifikasi kebutuhan perusahaan seputar sistem komputer
2. Mengidentifikasi ancaman dan tanggap terhadap isu yang berhubungan dengan sistem
3. Melakukan instalasi serta audit software dan hardware
Applications Developer
Bertanggung jawab terhadap pembuatan dan pengujian aplikasi yang didesain khusus untuk suatu perangkat. Applications developer biasanya memiliki spesialisasinya sendiri. Misal ada developer yang fokus pada app dengan sistem operasi iOS.

Tugas-tugas app developer antara lain:
1. Mengembangkan aplikasi untuk memenuhi kebutuhan pelanggan
2. Membuat dan menggunakan source code dalam sebuah aplikasi
3. Mengevaluasi aplikasi yang sudah ada dan membuat modifikasi
Mobile App Developer
Bertanggung jawab terhadap pembuatan dan pengujian aplikasi yang didesain khusus untuk perangkat mobile. Mobile app developer umumnya terdapat spesialisasi sistem operasi iOS dan Android.

Tugas-tugas mobile app developer antara lain:
1. Mengembangkan aplikasi sesuai sistem operasi atau spesialisasinya masing-masing
2. Melakukan inovasi untuk meningkatkan fungsionalitas dan menghadirkan fitur terbaru
3. Maintenance aplikasi secara berkala agar fungsionalitas aplikasi tetap terjaga
